# Quantium_Chip_Analysis
### Customer segmentation and purchasing behavior analysis for chips category using R


## Overview

This project analyzes customer purchasing behaviour in the chips category using transaction and customer data. The goal is to identify key customer segments and provide insights for category strategy.

## Tools Used

* R (tidyverse, ggplot2, dplyr)
* R Markdown
* Data cleaning & feature engineering

## Key Tasks

* Cleaned transaction and customer datasets
* Removed outliers and irrelevant products
* Extracted brand names and pack sizes
* Merged datasets for analysis
* Performed customer segmentation analysis
* Built visualisations for sales trends

## Key Insights

* Mainstream customers generated the highest chip sales
* 175g pack sizes were most popular
* Certain brands dominated premium segments
* Young Singles/Couples were the most valuable segment

## Recommendation

Focus marketing efforts on mainstream young customers and optimise product placement for 175g packs to increase category sales.
